# eink

For Pinout see: https://github.com/atc1441/ZBS_Flasher/blob/main/README.md

## requirements

### Packages
* sdcc
* Python3
* platformio
* esptool


## installation

### packages arch

```
yay -Syu sdcc platformio unzip
pip install esptool
```

### packages debian/ubuntu

```
sudo apt install sdccplatformio unzip
pip install esptool
```

### repo initialisation

Run ```init.sh``` to clone necessary repos

## build

run ```build.sh```
